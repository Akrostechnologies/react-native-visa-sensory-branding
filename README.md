# react-native-visa-sensory-branding

react native visa sensory branding

https://twitter.com/jamesjara

## Installation

```sh
npm install react-native-visa-sensory-branding
```

## Usage

```js
import VisaSensoryBranding from "react-native-visa-sensory-branding";
 
VisaSensoryBranding.show({
    checkMarkShown: true,
    soundEnabled: true,
});

```
